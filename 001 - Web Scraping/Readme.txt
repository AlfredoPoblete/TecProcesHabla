Primer trabajo practico de Web Scraping, aplicado a una pagina web relacionado con la historia de mi juego favorito: Castlevania Symphony of the Night.
Este proceso de extracción de contenido, se realizo con el fin de estudiar aspectos del mismo como:
1 - Mostrar todo el texto principal y su titulo
2 - Mostrar la cantidad de veces que se repite una determinada palabra.
3 - Generar un párrafo aleatorio y mostrar su longitud.
4 - Graficar una nube de palabras para resaltar la frecuencia con la que un término específico o categoría aparece en el texto principal.
5 - Descargar la imagen generada
