Cuaderno de Jupyter Notebook que contiene los diferentes scripts que podemos ejecutar utilizando la librería spaCy, biblioteca de procesamiento de lenguaje natural.
Utilizando un texto como prueba, utilizamos spaCy para:
1 - Tokenización
2 - Lematización.
3 - Análisis de dependencias.
